☆ [for portfolio] **(https://arcadetokyo.github.io/arcad3z-portfolio/)**

built with HTML, CSS, and vanilla JavaScript. 
bo frameworks or external libraries — lightweight by choice, with all interactions written from scratch.

☆ **how it's structured**

★ index.html — landing
★ projects.html — dev and cs projects
★ skills.html — skills, tools, experience, certifications
★ work.html — creative work across design, photography, copywriting, and video
★ contact.html — contact and socials

★ css/ — shared base styles plus one stylesheet per page
★ js/ — shared animation scripts and page-specific logic
★ assets/ — fonts, icons, images
★ projects/ — project media organized by category


☆ ** to run locally**

git clone https://github.com/arcadetokyo/arcad3z.git
cd arcad3z
python3 -m http.server 8000
